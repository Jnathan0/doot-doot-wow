Summary:

This project is a discord sound bot that allows users to upload and play sounds through the discord client by typing slash commands or text commands with a delimiter.
The project also allows users to upload images to pair with sounds when a sound is played

Project Technologies:

- python3.10+ runtime
- ffmpeg
- docker
- redis
- sqlite3

Libraries:

This project depends on the discord.py library for its interactions with the discord service api
